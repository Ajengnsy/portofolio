Materi yang Dipahami

HTML (Hypertext Markup Language) adalah bahasa markah standar untuk membuat dan merancang halaman web. Digunakan bersama dengan CSS (Cascading Style Sheets) dan JavaScript, HTML memberikan struktur dasar dari sebuah halaman web dengan menggunakan elemen-elemen seperti tag, atribut, dan nilai. Dengan HTML, pengembang web dapat menentukan bagaimana konten halaman web mereka diorganisir dan ditampilkan kepada pengguna, mulai dari teks, gambar, video, hingga elemen-elemen interaktif seperti formulir. Ini memungkinkan web untuk menjadi medium yang interaktif dan dinamis, memfasilitasi pertukaran informasi, komunikasi, dan kolaborasi di seluruh dunia melalui internet.

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain tampilan dan format halaman web yang ditulis dalam HTML. CSS memungkinkan pengembang web untuk mengontrol tata letak, warna, jenis huruf, ukuran, dan banyak aspek lainnya dari elemen-elemen HTML di dalam sebuah halaman web. Dengan menggunakan aturan dan properti CSS, pengembang dapat membuat halaman web lebih menarik, mudah dibaca, dan mudah digunakan oleh pengguna. CSS juga memungkinkan pemisahan antara struktur konten (HTML) dan tampilan (CSS), sehingga memudahkan perawatan dan pengembangan halaman web secara terpisah. Selain itu, CSS memiliki konsep kaskade, di mana aturan-aturan CSS dapat diwariskan, ditumpuk, dan ditimpa, memberikan fleksibilitas dan kontrol yang lebih besar dalam merancang tampilan halaman web.

Figma adalah platform desain berbasis web yang memungkinkan kolaborasi dalam pengembangan desain antarmuka pengguna (UI) dan desain antarmuka pengguna (UX). Dibandingkan dengan perangkat lunak desain tradisional yang berjalan di desktop, Figma memungkinkan tim desain untuk bekerja secara bersama-sama dalam waktu nyata, bahkan jika mereka berada di lokasi yang berbeda. Ini memfasilitasi kolaborasi yang lebih baik antara desainer, pengembang, dan pemangku kepentingan lainnya dalam proses pembuatan desain. Figma menawarkan berbagai fitur, termasuk pembuatan prototipe interaktif, pengelolaan komponen yang mudah, pembagian tugas, serta integrasi dengan alat pengembangan lainnya. Platform ini semakin populer karena kemudahan penggunaannya, skalabilitas, dan kemampuannya untuk mendukung kolaborasi tim secara efisien.

Materi yang belum dipahami

JavaScript adalah bahasa pemrograman yang digunakan untuk menambahkan interaktivitas dan fungsionalitas dinamis ke dalam halaman web. Dengan sintaks yang mirip dengan bahasa pemrograman lain dan kemampuannya untuk dijalankan di browser pengguna, JavaScript memungkinkan pengembang untuk membuat aplikasi web yang responsif dan dinamis tanpa perlu mengirimkan permintaan server berulang kali, menjadikannya bahasa yang sangat penting dalam pengembangan web modern.

PHP adalah singkatan dari "Hypertext Preprocessor", adalah bahasa pemrograman sisi server yang digunakan untuk mengembangkan situs web dinamis dan aplikasi web. PHP terintegrasi dengan HTML, memungkinkan pengembang untuk menanamkan kode PHP di dalam dokumen HTML untuk menghasilkan konten dinamis. PHP dapat berinteraksi dengan berbagai jenis basis data, membuatnya cocok untuk membangun aplikasi web yang melibatkan pengelolaan data seperti situs e-commerce, forum, dan sistem manajemen konten. Karena sifatnya yang open source dan mudah dipelajari, PHP menjadi salah satu bahasa pemrograman web yang paling populer dan banyak digunakan di dunia.

Database adalah kumpulan data yang terstruktur dan terorganisir secara sistematis dalam satu tempat yang dapat diakses dan dikelola dengan menggunakan perangkat lunak tertentu. Database memungkinkan pengguna untuk menyimpan, mengambil, dan memanipulasi data dengan efisien. Ada berbagai jenis database, termasuk database relasional yang menggunakan tabel untuk menyimpan data dengan keterkaitan antar data, serta database non-relasional seperti database NoSQL yang menyimpan data dalam bentuk dokumen atau grafik. Penggunaan database sangat luas, mulai dari pengelolaan informasi dalam aplikasi bisnis, sistem perbankan, aplikasi web, hingga penyimpanan data ilmiah dan penelitian. Dengan kemampuannya untuk menyediakan akses cepat dan aman ke data, database memainkan peran kunci dalam pengembangan aplikasi modern dan pengelolaan informasi.
